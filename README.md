# صفحات التحويل لمجموعات التلغرام

هذه الصفحات تحول المستفيد تلقائياً لمجموعة التلغرام الخاصة بدورته خلال 3 ثوانٍ.

## الملفات:
- a1.html → إدارة الأزمات والتعامل مع الحشود
- a2.html → خدمة العملاء في وزارة الداخلية
- a3.html → ضمانات وحقوق المتهم في مرحلة التحقيق
- a4.html → مبادئ الذكاء الاصطناعي الأمني
- a5.html → إدارة الموارد البشرية
- a6.html → المحادثة في اللغة الإنكليزية المستوى الأول
- a7.html → القيادة وإدارة الفرق الأمنية
- a8.html → الرخصة الدولية لقيادة الحاسب الآلي (ICDL)
